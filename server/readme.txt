-- eureka server 特性：1）失效剔除：默认60秒将当前清单中超过90秒的没有续约的服务剔除出去。2）自我保护：如果eureka server在运行期间会将"心跳"失败比例在15分钟内低于85%的注册信息保护起来，让其不过期。但是客户端可能会出现拿到不存在的服务实例导致失败。eureka.server.enableSelfPreservation=true  改为false则确保不可用的服务剔除3）region(地域) 和 zone（可用区）region : 地域 ------>  zone : 地域的机房             (一对多)region默认配置为：us-east-1，zone默认配置为：defaultZoneeureka server通过eureka.client.serviceUrl.defaultZone属性设置eureka的服务注册中心位置
